# Beitragen zum Business Trust Circle

Dieses Repo ist ein lebendiges Dokument – Vorschläge und Verbesserungen sind ausdrücklich willkommen!

## Wie du beitragen kannst

### Ideen & Vorschläge einbringen

Erstelle ein **Issue** auf GitHub:

1. Gehe auf die [Issues-Seite](../../issues) des Repos
2. Klicke auf "New Issue"
3. Beschreibe deinen Vorschlag – gerne auch formlos und in Stichpunkten
4. Wir diskutieren gemeinsam in den Kommentaren

Das ist der einfachste Weg, Ideen einzubringen – auch ohne technisches Vorwissen.

### Änderungen am Text vorschlagen

Wenn du eine konkrete Textänderung vorschlagen möchtest, erstelle einen **Pull Request**:

1. Forke das Repo (Button oben rechts)
2. Bearbeite die gewünschte Datei direkt auf GitHub (Stift-Symbol)
3. Beschreibe kurz, was du geändert hast und warum
4. Erstelle den Pull Request

GitHub macht es einfach, Textdateien direkt im Browser zu bearbeiten – kein Programmieren nötig.

## Leitlinien

- **Respektvoller Umgang** – Der [Wertekatalog](KONZEPT.md#2-wertekatalog) gilt auch hier
- **Begründe deine Vorschläge** – Ein kurzes "Warum?" hilft bei der Diskussion
- **Kleine Änderungen** sind genauso willkommen wie große Ideen

## Fragen?

Erstelle einfach ein Issue mit deiner Frage – es gibt keine dummen Fragen.
